amux

Agent Multiplexer

Run dozens of Claude Code agents in parallel — monitored, controlled, and coordinated from a single dashboard. Access everything from your browser, phone, or terminal.

Install GitHub Docs
Open source  ·  MIT License  ·  github.com/mixpeek/amux
Install & quickstart
$ git clone https://github.com/mixpeek/amux && cd amux && ./install.sh

# Register a project and start an agent
$ amux register api --dir ~/Dev/api --yolo
$ amux start api

# Open the web dashboard (or install as a PWA on your phone)
$ amux serve
  → https://localhost:8822

Capabilities

  • Parallel agents — register any number of Claude Code sessions, each in its own tmux pane, all visible from one place. Start, stop, and send commands to any agent without switching terminals.
  • Web dashboard + PWA — full-featured HTTPS dashboard installable as a home screen app on iOS and Android. Monitor agents, send commands, and browse output from your phone.
  • Live peek & control — scroll through any agent's full output, search it, send slash commands, and use quick-action chips (Ctrl-C, /compact, /clear) without leaving the dashboard.
  • File attachments — send files directly to any agent. Paste an image with Ctrl-V, drag and drop onto the send bar, or click the 📎 button. Supports images, PDFs, text, CSV, JSON, and more.
  • Built-in kanban board — track work across all agents in one place. Auto-generated issue keys (PROJ-1), tags, due dates, drag-and-drop columns, calendar view, and an iCal feed. Backed by REST API.
  • REST API — every action is an endpoint. Automate agent orchestration with curl or any HTTP client. Register, start, stop, send, query status, and manage board items programmatically.
  • Session & global memory — each agent gets its own persistent memory file, plus a shared global memory. Edit directly in the dashboard and inject context into any session.
  • Offline-first — IndexedDB + Service Worker keep sessions browsable without connectivity. Commands sent offline are queued and replayed automatically on reconnect via Background Sync.
  • Token stats — daily token usage across every session with per-agent breakdowns and bar charts. Know exactly what each agent is spending before your bill arrives.
  • Zero dependencies — single Python file, no build step. Requires only Python 3 and tmux. Install in seconds, run anywhere.
